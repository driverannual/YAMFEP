YAMFEP is an easy-to-use Photoshop script that activates specific layers by name (can be text, photos, or shapes), exports them as JPG, and deactivates them. You can automate repetitive tasks like creating email signatures for 50 people, or anything you need, saving valuable time.
# Download
Go to the [releases page](https://github.com/driverannual/YAMFEP/releases/) and download the ZIP file. So you can export in any local you want.

### How to use
1. Go to File > Scripts > Browse;
2. Run 'script.jsx';
3. Select the layers you want to export individually, unselect the permanent ones (like backgrounds or logos).

That's all! The files will be saved in 'output', next to your PSD.

Easy as it should be.
